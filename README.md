# TypingSpeedTest
## Project Description
Typing Speed Test is a Python application that measures typing speed, accuracy, errors, and time taken. It uses keyboard events, time measurement, string comparison, and Tkinter for an interactive user interface.
## Features
- Measures typing speed
- calculate typing accurancy
- display time taken
- simple and easy to use interface
- restart the test
## Technologies Used
- Python
- Tkiter
## How to run
1. Install python
2. open 'main.py' in thonny
3. click run
4. start typing the given test
5. view your typing speed and accurancy.
